## 快速开始

### 使用活动套餐前置工具

1. 打开 `活动套餐前置.html`
2. 上传包含以下字段的Excel文件:
   - 单据编号
   - 客户
   - 规格型号
   - 促销员  
   - 销售数量
   - 大类型(品类)

3. 完成字段映射
4. 依次执行:
   - 旧套餐计算
   - 套餐政策计算
   - 活动机型政策计算

5. 导出结果,导入主提成计算工具继续处理

### 使用六合一提成计算工具

1. 打开 `六合一提成计算.html`
2. 上传数据文件
3. 按顺序执行六个模块:
   - ① 特价机
   - ② 壁挂炉
   - ③ 销售政策
   - ④ 烟灶
   - ⑤ 国补提成
   - ⑥ 折扣机(前五个模块完成后解锁)

4. 导出Excel结果
