# 政策配置快速入门指南 🚀

> 5分钟快速上手政策配置功能

## 第一次使用？跟着这3步走！

### 步骤 1️⃣：下载模板

有两种方式下载模板，任选其一：

**方式A：使用独立生成器**（推荐新手）
1. 打开 `生成政策模板.html` 文件
2. 点击"下载政策配置模板"按钮
3. 保存 `活动套餐政策配置模板.xlsx` 文件

**方式B：从主工具下载**
1. 打开 `活动套餐前置.html`
2. 在顶部"政策配置文件"区域
3. 点击"点击下载Excel模板"链接

---

### 步骤 2️⃣：编辑模板

用Excel或WPS打开下载的模板文件：

#### 📊 Sheet 1: 旧套餐规则

**这是什么？** 固定产品组合的套餐，如"烟机+灶具套装"

**快速上手：**
```
如果要添加新套餐，新增一行：
套餐编号 | 产品1         | 价格1 | 产品2      | 价格2 | 套餐价 | 提成金额
8       | 新产品A       | 5000  | 新产品B    | 3000  | 7500   | 400
```

**注意：** 不用的列留空即可

---

#### 🧩 Sheet 2: 组合套餐政策

**这是什么？** 基于品类的组合，如"烟机+热水器+净水"

**快速上手：**
```
修改九件套的激励从1000改为1200：
找到"九件套"那一行，把"单套激励"列改为 1200
```

**添加新组合：**
```
套餐名称 | 优先级 | 单套激励 | 必需品类1 | 必需品类2 | 特殊说明
三件套   | 6     | 200     | 烟机      | 净水      |
```

**优先级规则：** 数字越小越优先（1最高，2次之...）

---

#### 🔥 Sheet 3: 活动机型政策

**这是什么？** 特定机型的专项激励

**快速上手 - 直接加提型：**
```
机型型号     | 类型 | 激励模式 | 单台激励 | 阈值 | 说明备注
新机型XXX   | 燃热 | flat    | 150     |     | 加提150元/台
```

**快速上手 - 阶梯制：**
```
机型型号     | 类型 | 激励模式   | 单台激励 | 阈值 | 说明备注
新机型YYY   | 净水 | threshold | 100     | 5   | 满5台每台100元
```

**激励模式说明：**
- `flat` = 每台都给钱，无门槛
- `threshold` = 达标才给钱，全量计算

---

### 步骤 3️⃣：上传使用

1. 保存编辑好的Excel文件
2. 打开 `活动套餐前置.html`
3. 在"政策配置文件"区域点击上传或拖拽文件
4. 看到"政策配置加载成功并已保存"提示 ✅
5. 继续正常使用工具即可

**🎉 配置自动保存！** 下次打开页面会自动加载您的配置，无需重复上传！

## 💡 实用技巧

### ✅ 必学技巧

1. **先下载模板再改**  
   不要从空白Excel开始，容易格式错误

2. **保留原有数据**  
   如果只想改一项，其他的保持原样

3. **测试后再用**  
   先用小量数据测试，确认无误再正式使用

### 🔥 高级技巧

1. **保存多个版本**  
   不同时期的政策保存为不同文件，如：
   - `政策配置_9月.xlsx`
   - `政策配置_10月.xlsx`

2. **优先级调整**  
   如果发现某个套餐总是没匹配上，提高它的优先级（改小数字）

3. **型号要准确**  
   型号大小写要跟实际数据一致，建议复制粘贴

## 🎯 常见场景示例

### 场景1：增加新的旧套餐

**需求：** 新增"型号A + 型号B"套餐，提成500元

**操作：**
1. 打开Sheet 1（旧套餐规则）
2. 在最后一行下方新增一行
3. 填写：
   ```
   8 | 型号A | 6000 | 型号B | 4000 | | | | | 9500 | 500
   ```
4. 保存并上传

---

### 场景2：修改活动机型激励

**需求：** JSQ31-VJSAI 的激励从100改为150

**操作：**
1. 打开Sheet 3（活动机型政策）
2. 找到"JSQ31-VJSAI"那一行
3. 把"单台激励"列从 100 改为 150
4. 保存并上传

---

### 场景3：添加新的活动机型

**需求：** 新机型 "XYZ-2000" 参加活动，每台加提200元

**操作：**
1. 打开Sheet 3（活动机型政策）
2. 在最后一行下方新增：
   ```
   XYZ-2000 | 热水器 | flat | 200 | | 新品加提200元/台
   ```
3. 保存并上传

## ❓ 常见问题 FAQ

### Q1: 配置文件上传后，刷新页面还在吗？ ✨
**A:** **在！** 配置会自动保存在浏览器中，刷新页面、关闭浏览器后重新打开都会自动加载。

### Q2: 下次打开需要重新上传配置吗？
**A:** **不需要！** 配置自动保存，下次打开会自动加载，一次配置长期有效。

### Q3: 如何恢复默认配置？
**A:** 在配置状态栏中点击"恢复默认"链接，或者清除浏览器缓存。

### Q4: 能不能只改一部分？
**A:** 不能。上传会替换全部规则，所以要保留不改的部分。

### Q5: 修改配置后要重新上传数据吗？
**A:** 不用。直接重新点击计算按钮即可。

### Q6: 如何知道配置是否生效？
**A:** 上传成功会显示绿色提示"政策配置加载成功并已保存"，且页面上的政策说明会实时更新。

### Q7: 换电脑后配置还在吗？
**A:** 不在。配置保存在当前浏览器的本地存储中，换电脑或换浏览器需要重新上传。

### Q8: Excel打不开怎么办？
**A:** 确认文件后缀是 .xlsx，可以用WPS或在线Excel打开。

## 🆘 遇到问题？

### 错误信息对照表

| 错误提示 | 可能原因 | 解决方法 |
|---------|---------|---------|
| 政策文件解析失败 | Sheet名称错误 | 检查是否为"旧套餐规则"等 |
| 无法读取文件 | 文件格式不对 | 确认是.xlsx或.xls格式 |
| 数据格式错误 | 数值字段填了文字 | 检查价格、金额等是否为数字 |

### 检查清单

上传前过一遍这个清单：

- [ ] Sheet名称是否正确？
  - ✅ 旧套餐规则
  - ✅ 组合套餐政策  
  - ✅ 活动机型政策

- [ ] 列名是否保持不变？
  - [ ] 没有删除列
  - [ ] 没有修改列名
  - [ ] 没有调整列顺序

- [ ] 数值字段是否为纯数字？
  - [ ] 价格、套餐价、提成金额
  - [ ] 优先级、单套激励
  - [ ] 单台激励、阈值
