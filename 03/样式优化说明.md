# 03 模块样式优化说明

## 优化概述

本次优化使 03 文件夹中的工具页面与首页设计风格保持一致,并提取 CSS 以便维护。

## 主要改动

### 1. 创建统一样式文件
- 新增 `03/styles.css` - 包含与首页一致的设计系统
- 使用深蓝金黄配色方案 (--accent-blue / --accent-gold)
- 定义完整的 CSS 变量系统(颜色、间距、圆角、阴影等)
- 提供响应式设计支持

### 2. 更新 HTML 文件

#### 活动套餐前置.html
- ✅ 引用外部样式文件 `styles.css`
- ✅ 添加返回首页导航按钮
- ✅ 更新标题和元信息
- ✅ 简化内联样式,仅保留页面特定样式
- ✅ 使用统一的容器类名 (.tool-container, .tool-header, .tool-content)
- ✅ 使用 CSS 变量替代硬编码颜色值

#### 六合一提成计算.html
- ✅ 引用外部样式文件 `styles.css`
- ✅ 添加返回首页导航按钮
- ✅ 新增页面头部区域 (.page-header)
- ✅ 简化样式,使用 CSS 变量
- ✅ 保留特有的功能组件样式(模块按钮、数据表格等)

## 设计系统一致性

### 颜色方案
```css
/* 品牌色 */
--accent-blue: #001F5C;     /* 主色调 - 深蓝 */
--accent-gold: #FFC107;      /* 强调色 - 金黄 */

/* 背景色 */
--bg-primary: #F8F9FA;
--bg-secondary: #E9ECEF;
--bg-card: #FFFFFF;

/* 文字颜色 */
--text-primary: #001F5C;
--text-secondary: #003D7A;
--text-tertiary: #6C757D;

/* 功能色 */
--color-success: #10B981;
--color-warning: #F59E0B;
--color-danger: #EF4444;
--color-info: #3B82F6;
```

### 组件风格
- 统一的按钮样式(悬停效果、禁用状态)
- 一致的表单元素设计
- 统一的卡片、表格、提示框样式
- 规范的阴影和圆角使用

## 优势

### 1. 维护便利
- CSS 集中管理,修改一处即可影响所有页面
- 变量系统使颜色、间距调整更方便
- 代码复用,减少重复

### 2. 用户体验
- 视觉风格统一,品牌识别度强
- 响应式设计,支持各种屏幕尺寸
- 流畅的交互动画

### 3. 开发效率
- 新页面可快速套用统一样式
- 减少样式冲突
- 更易于团队协作
