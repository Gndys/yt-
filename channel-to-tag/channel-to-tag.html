常见问题

关于YouTube频道标签提取工具的常见疑问解答

为什么需要API密钥？

我们的工具使用YouTube Data API v3来获取频道和视频数据。由于YouTube的API限制，您需要提供自己的API密钥才能使用此工具。这样可以确保工具的稳定性和可用性。

如何获取YouTube API密钥？

您可以在Google Cloud Console创建一个项目，启用YouTube Data API v3，然后创建API密钥。详细步骤可以在我们的"管理API密钥"页面中找到。

工具可以分析多少个视频？

默认情况下，工具会分析频道最近的50个视频。这通常足以获取频道的主要标签和关键词趋势。

我的API密钥安全吗？

是的，您的API密钥仅存储在您的浏览器本地，不会上传到我们的服务器。所有分析都在您的浏览器中完成，确保您的API密钥安全。

支持哪些类型的YouTube频道链接？

我们支持多种格式的YouTube频道链接，包括标准频道链接(youtube.com/channel/UC...)、自定义URL(youtube.com/c/...)、用户名链接(youtube.com/user/...)和新的@handle格式(youtube.com/@...)。

导出的数据包含哪些内容？

导出的数据包括频道信息(如名称、订阅数等)、提取的标签和关键词(按使用频率排序)，以及分析的视频列表。Excel格式导出包含多个工作表，提供更详细的数据。
