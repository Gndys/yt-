## 🎨 设计系统核心

### 颜色系统
```css
/* 品牌色 */
--accent-blue: #001F5C;      /* 深蓝 - 主色调 */
--accent-gold: #FFC107;      /* 金黄 - 强调色 */

/* 背景色 */
--bg-primary: #F8F9FA;       /* 主背景 */
--bg-secondary: #E9ECEF;     /* 次背景 */
--bg-card: #FFFFFF;          /* 卡片背景 */

/* 功能色 */
--color-success: #10B981;    /* 成功 - 绿 */
--color-warning: #F59E0B;    /* 警告 - 黄 */
--color-danger: #EF4444;     /* 危险 - 红 */
--color-info: #3B82F6;       /* 信息 - 蓝 */

/* 文字色 */
--text-primary: #001F5C;     /* 主文字 */
--text-secondary: #003D7A;   /* 次文字 */
--text-tertiary: #6C757D;    /* 辅助文字 */
```

### 组件规范

#### 按钮
- `btn btn-primary` - 主要操作（金黄色）
- `btn btn-secondary` - 次要操作（深蓝色）
- `btn btn-success` - 确认操作（绿色）
- `btn btn-danger` - 危险操作（红色）
- `btn btn-outline` - 轮廓按钮

#### 尺寸
- `btn-sm` - 小按钮
- 默认 - 中等按钮
- `btn-lg` - 大按钮

### 布局规范

#### 容器
- `tool-container` - 工具主容器
- `tool-header` - 头部容器
- `tool-content` - 内容容器
- `tool-footer` - 页脚容器

#### 区块
- `section` - 内容区块
- `section-header` - 区块头部
- `section-title` - 区块标题
- `section-description` - 区块描述

## 📝 注意事项

### 更新时必须检查
- ✅ 功能是否正常
- ✅ 样式是否正确
- ✅ 交互是否流畅
- ✅ 响应式是否正常
- ✅ 浏览器兼容性

### 常见问题

**Q: 更新后样式不生效？**
A: 检查样式文件路径是否正确，清除浏览器缓存

**Q: 原有功能失效？**
A: 检查是否错误修改了 JavaScript 代码或元素 ID

**Q: 移动端显示异常？**
A: 检查是否使用了固定宽度，建议使用响应式单位

**Q: 颜色与设计系统不符？**
A: 使用 CSS 变量而非硬编码颜色值
