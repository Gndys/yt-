# A.O.史密斯业务工具平台 - 样式统一指南

## 📋 概述

本指南说明如何将所有工具页面统一到新的设计系统。

## 🎨 设计系统

### 配色方案
- **主色调**: 深蓝色 (#001F5C)  
- **强调色**: 金黄色 (#FFC107)
- **背景色**: 浅灰色 (#F8F9FA)
- **功能色**: 成功(绿) / 警告(黄) / 危险(红) / 信息(蓝)

### 核心组件
所有样式定义在 `common-styles.css` 中，使用 CSS 变量系统。

## 🔧 应用步骤

### 1. 引入样式文件

**替换旧的样式引用：**
```html
<!-- 旧的 -->
<link rel="stylesheet" href="styles.css">

<!-- 新的 -->
<link rel="stylesheet" href="../common-styles.css">
```

## 💡 最佳实践

1. **保持一致性**: 使用统一的 class 命名，不要混用内联样式
2. **响应式设计**: 所有组件已适配移动端
3. **渐进增强**: 确保基础功能在老旧浏览器也能工作
4. **性能优化**: 避免不必要的 DOM 操作和重绘
5. **可访问性**: 确保键盘导航和屏幕阅读器支持
